# CheeSweeper

This is a basic Minesweeper game written in Python 3 using tkinter as a graphics library.


## Requirements

* Python 3 (tested on 3.5.2)


## Usage

To run the game, execute:

```
./cheesweeper.py
```

Field dimensions and number of mines are hardcoded.
